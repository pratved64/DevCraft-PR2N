# DevCraft-PR2N

## Flow
Student visits stall
-> Student makes a purchase at stall
-> Student scans the "fest passport"
-> Student metrics are tracked via the app

## Sponsor Metrics:
* Total Footfall
* Peak Traffic times
* % of users that scanned at stalls (distribution)
* Anonymous target demographics
* Cost per interaction (total package / checked in students)
* Cross Pollination Rate from other stalls

## Tech Stack
* Next.js | next-pwa or @serwist/next
* MongoDB + IndexedDB for offline local db on mobile
* Flask/FastAPI microservice for Fraud Detection/Crowd Heatmaps feature
